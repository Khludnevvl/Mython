## MyThon

Учебный проект, созданный для понимания работы компиляторов и интерпретаторов.

Простой интерпретатор языка Mython (mini-python).
Поддерживаются условные операторы, классы и наследование.



---
#### Сборка

> При сборке CMake-ом, возможно собрать программу с тестами (прогоняет большое количество тестов из каталога tests, пользовательский ввод не доступен). Для этого при сборке указать ключ -DBUILD_TESTS=ON.

---

### Использование:

> ./mython input_file output_file 

Вводится имя входного файла с кодом Mython. Результатом является вывод функции print в output_file.

Пример функции print:
>x = 4
w = 'world'
print x, x + 6, 'Hello, ' + w  // Выведет 4 10 Hello, world 

Пример входного и выходного файлов находится в каталоге examples.

---
#### Поддерживаемые типы :

**Числа**

Могут использоваться только целые числа. Возможные арифмитические операции: сложение, вычитание, умножение, целочисленное деление.

**Строки**

Последовательность символов, размещенная на одной строке и ограниченная двойными или одинарными ковычками. Неизменяемые.

**Bool и None**

Поддерживаются логические значения True и False. Есть также специальное значение None.

**Классы**

Объявление класса начинается с ключевого слова class, за которым следует идентификатор имени и объявление методов класса. 

---
### Операции:

В Mython определены:
* Арифметические операции для целых чисел. Деление на ноль вызывает ошибку времени выполнения.
* Сложение строк, например: s = 'hello, ' + 'world'
* Операции сравнения строк и целых чисел: ==, !=, <=, >=, <, >.
* Логические операции and, or, not.
* Унарный минус.
* Операция сложения применима к классам с методом \__add __.
* Операции сравнения применяются не только к числам и строкам, но и к объектам классов, имеющих методы \_\_eq__ (проверка «равно») \_\_lt__ (проверка «меньше»).
